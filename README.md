# The Maze — What Type of AI User Are You?

A shareable quiz game based on Chapter 13 ("The Maze — Four Ways People Get Stuck")
of **Stop Wasting Time With AI** by Zena Collins.

- Four types: Firestarter 🔥 · Grounder 🌱 · Ocean 🌊 · Fresh Air 💨
- Static site (single HTML file + character images), served by nginx
- Characters courtesy of [Feel-Wise.com](https://feel-wise.com)
- Part of the SWTWAI TikTok campaign — quiz is the link-in-bio funnel

Deployed on Dailey OS.
